Ce site a été developpé par Florent Kirch et Morgane Julien, dans le cadre d'un projet de programmation Web pour l'IUT informatique de Bordeaux. Il a été développé en HTML - CSS - PHP et un peu de Javascript.

Dans cette partie, nous parlerons des problèmes que nous avons rencontrés lors de la création de ce site Le premier vrai problème que nous avons rencontré a été de lier correctement ce site à la base de donnée que l'IUT nous fournis. Ensuite, dans l'affichage du contenu de la Base de Donnée, nous avons rencontrés certains problèmes : notamment pour programmer le fait qu'en cliquant sur les photos de chaque compositeur, on soit redirigé vers une page contenant tous les albums de ce compositeur. La liaison avec l'API D'Amazon a été quelque peu compliqué, notamment a cause du fait que c'était la première fois que nous devion intégré à un site une API complète et professionnelle. On à d'abord commencé par comprendre l'API pour ensuite essayer de l'adapter à notre site pour y mettre un lien redirigeant vers la page Amazon adaptée.
